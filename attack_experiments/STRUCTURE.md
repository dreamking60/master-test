## ğŸ“‚ Directory Purposes

### `scripts/` - Executable Scripts
All Python and shell scripts for running experiments.

**Core Scripts:**
- `normal_controller.py` - Simulates legitimate user
- `injection_attack.py` - Performs injection attack
- `record_trajectory.py` - Records robot position
- `analyze_trajectory.py` - Analyzes recorded data

**Experiment Scripts:**
- `quick_test.sh` - Quick test without recording
- `run_injection_experiment.sh` - Interactive experiment
- `run_experiment_with_recording.sh` - Full experiment with recording

### `docs/` - Documentation
All markdown documentation files.

**Main Docs:**
- `README_EN.md` - Complete guide
- `INJECTION_EXPERIMENT.md` - Experiment details
- `LOGIC_AND_SUCCESS_CRITERIA.md` - Technical explanation
- `EXPERIMENT_RESULTS.md` - Results documentation

### `data/` - Trajectory Data
CSV files containing recorded robot positions.

**File Format:**
- Naming: `trajectory_YYYYMMDD_HHMMSS.csv`
- Contains: timestamp, position (x,y,z), orientation, velocities
- Generated by: `record_trajectory.py`

**Usage:**
- Input for: `analyze_trajectory.py`
- Can be opened in: Excel, Python pandas, etc.

### `results/` - Analysis Results
Generated plots and analysis outputs.

**File Types:**
- PNG plots: Trajectory visualizations
- Generated by: `record_trajectory.py` (with --plot flag)
- Used for: Reports, presentations
